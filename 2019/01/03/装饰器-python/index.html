什么是装饰器

python装饰器就是用于拓展原来函数功能的一种函数，目的是在不改变原函数定义的情况下，给函数增加新的功能。
这个函数的特殊之处在于它的返回值也是一个函数，这个函数是内嵌“原”函数的函数

在代码运行期间动态的增加功能
装饰器(decorator)是修改其它函数功能的函数,是返回函数的高阶函数
