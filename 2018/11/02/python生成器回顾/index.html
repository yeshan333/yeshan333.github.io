python生成器（generator）

  • 生成器是一种使用普通函数语法定义的迭代器
  • 包含yield语句的函数都是生成器，它是一个不断产生值的函数
  • 生成器每次使用yield产生一个值后，函数都将冻结，即在此处停止执行，等待重新被唤醒。被唤醒后从停止的地方开始继续执行

生成器推导（生成器表达式）

* 使用圆括号()创建一个生成器推导 *,它创建了一个可迭代的对象
使用next()函数可以获得生成器推导的下一个返回值

g = (i**2 for i in range(10))
